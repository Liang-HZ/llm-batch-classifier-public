# Code of Conduct

## Our Commitment

We want this project to be a respectful, technically useful place for contributors and users.

Examples of behavior that helps:

- Giving clear, specific, and respectful technical feedback
- Assuming good intent while still being rigorous
- Sharing context, repro steps, and evidence when raising issues
- Respecting different levels of experience

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks
- Publishing private information without permission
- Deliberate disruption, trolling, or bad-faith arguments
- Sexualized language or imagery in project spaces

## Scope

This Code of Conduct applies to repository issues, pull requests, reviews, and other project-managed spaces.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

Repeated or severe violations may lead to temporary or permanent exclusion from project spaces.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainer through the contact options available on the maintainer's GitHub profile.

When reporting, include enough detail for the maintainer to investigate.
